# Lab Activity 5 — Web Page using Bottle Framework

## Aim
Create a simple web app using the Bottle framework.

## Install Requirements (one-time, at repo root)
```bash
pip install -r requirements.txt
```

## Steps and Commands (after cloning)
```bash
cd LabActivity05_Bottle_Hello_World
python bottle_app.py
```

## Test in Browser
Open:
- http://localhost:8080/
