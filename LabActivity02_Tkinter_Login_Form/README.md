# Lab Activity 2 — Login Form using Tkinter (Grid Layout)

## Aim
Design a simple login form with Labels and Entry widgets arranged using the **Grid** geometry manager.

## Objective
- Create Labels, Entry fields, and Buttons
- Arrange widgets using `grid()`
- Collect username/password input
- Display a success/error message (demo validation)

## Steps and Commands (after cloning)
```bash
cd LabActivity02_Tkinter_Login_Form
python login_form.py
```
