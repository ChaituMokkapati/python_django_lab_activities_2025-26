# Lab Activity 1 — Tkinter Window (GUI App)

## Aim
Create a basic GUI application in Python using Tkinter that opens a window, sets a title, and displays a label.

## Objective
- Import the Tkinter package
- Create a root (main) window
- Set the window title and size
- Add a `Label` widget
- Run the Tkinter event loop

## Prerequisites
- Python 3.x installed (Tkinter comes with standard Python)

## Steps and Commands (after cloning)
```bash
# From repository root (environment already created/activated)
cd LabActivity01_Tkinter_Window
python window_gui.py
```
