# Lab Activity 4 — Hello World using Flask

## Aim
Create a simple Flask web app that shows **Hello, World!** in the browser.

## Install Requirements (one-time, at repo root)
```bash
pip install -r requirements.txt
```

## Steps and Commands (after cloning)
```bash
cd LabActivity04_Flask_Hello_World
python app.py
```

## Test in Browser
Open:
- http://127.0.0.1:5000/
