# Lab Activity 3 — Web Scraping using BeautifulSoup4

## Aim
Scrape data from a given URL using **requests** and **BeautifulSoup4**.

## Objective
- Fetch HTML content using `requests.get()`
- Parse HTML using BeautifulSoup
- Extract target elements (quotes + authors in this example)
- Print the extracted results

## Install Requirements (one-time, at repo root)
```bash
pip install -r requirements.txt
```

## Steps and Commands (after cloning)
```bash
cd LabActivity03_Web_Scraping_BeautifulSoup
python web_scraper.py
```
